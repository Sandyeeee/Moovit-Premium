# Moovit AdFree 🚍✨

[עברית](README_HE.md) | **English**

Tired of intrusive, full-screen advertisements ruining your user experience and making you miss your bus? This modified version removes all annoying ads from the Moovit application, giving you a clean, smooth, and fast interface.

---

## 📱 Compatibility
* **Android Only:** This modification is only compatible with Android devices.

---

## 🛠️ Installation Guide

Follow these simple steps to install the ad-free version:

1. **Download ReVanced GmsCore:**
   Go to the official [ReVanced GmsCore GitHub Repository](https://github.com/ReVanced/GmsCore) and download the latest APK from the **Releases** section. Install it on your device (this is required for the app to run properly).

2. **Download the App APK:**
   Navigate to the **Releases** section of this repository and download the latest modified Moovit APK file.

3. **Install & Enjoy:**
   Install the downloaded APK file on your Android device, open the app, and enjoy a completely ad-free experience!
